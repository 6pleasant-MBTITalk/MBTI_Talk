# MBTI matching social app - M친톡

## 프로젝트 소개
MBTI 친구 톡의 줄임말인 M친톡은 MBTI를 이용해서 자신의 MBTI와 상성이 잘 맞는 친구나 이성을 매칭을 시켜주는 소셜 앱입니다.
